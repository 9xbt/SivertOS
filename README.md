# SivertOS
A small, hobbyist 64-bit operating system

> [!NOTE]
> SivertOS is in a very early stage of development and it's not recommended to use it

### Features
- [X] Limine Boot Protocol
- [X] GDT
- [ ] IDT
- [ ] PS/2 driver
    - [ ] Keyboard
    - [ ] Mouse
- [ ] Serial driver
- [ ] ATA driver
- [ ] RTC driver
- [ ] GUI (someday)
