# SivertOS
A small hobbyist 64-bit operating system

> [!NOTE]
> SivertOS is in a very early stage of development, so it's not recommended for use on real hardware.

### Features
- [X] GDT
- [X] IDT
- [X] PS/2 driver
    - [X] Keyboard
    - [X] Mouse
- [X] Serial driver
- [X] RTC driver
- [X] Formatted printing (printf)
- [X] PMM
- [X] VMM
- [X] Heap
- [ ] IDE/ATA driver
